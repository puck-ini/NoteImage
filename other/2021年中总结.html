# 2021年中总结

这半年来感觉跟去年相比差距不大，但是又感觉不一样，学习方面还是回去学，迷茫也是真的迷茫。不知道以后该怎么办，不知道以后该做什么，学习东西还是零零散散做不到系统学习，学习的时候一想到自己学的零散知识过不久就会忘又没动力学下去了。苦逼。

# 工作

过年前工作上基本没有什么事情，过年后因为同事的离职，项目转手到我这里之后事情就开始变多了，不过跟去年相比感觉也没差多少，还是对项目进行修修补补，只是更频繁了。

# 学习

去年的计划是多看书，多写代码，多做笔记，先看看完成情况吧。

  • 读过的书

redis 设计与实现、effective java、领域驱动设计 、码出高效、实现领域驱动设计、博弈心理学、改变你一生的心理技巧、图解HTTP、漫画算法、java 多线程实战指南设计模式篇、计算机网络（谢希仁版）、java 阿里巴巴开发手册。

  • 写代码

写代码方面不算有什么进展。写了一个简单的 rpc 框架、看了 xxl-job 源码之后实现了一个简单的分布式任务调度系统、使用拦截器和责任链模式实现了一个简易网关、整理了之前学习的时候写的 demo等。最近也是在重构之前写过的项目。

  • 笔记

写笔记方面，一直以来我都是在有道云笔记上写的，不过写得不多，主要 markdown 写得不习惯。之前学习的时候在掘金上看过很多文章，但是不知道为什么就是不想在掘金上记录自己的笔记，之后在同事的推荐下选择了语雀。语雀的界面还是挺好看的，不过也没让我多写几篇笔记，半年来就写了两篇笔记，一个是Redis 学习笔记，，另一个是 Spring Data JPA 笔记，主要记录下之前自己常用的方法。之后机缘巧合下看到了 github pages 和 vuepress，就自己按照别人的教程弄了属于自己的在 github 上的博客。毕竟是自己搭建的博客，趁着兴奋劲连续写了几篇关于 Redis 的笔记，之前在语雀上写的笔记算是对 Redis 的一个入门，类似于一个大纲，这次在 github 上写的是对大纲里面的知识较详细的记录。

  • 看源码

这个算是去年计划之外的计划了，自己对源码这方面基本就是感兴趣什么就看什么。

java 源码：主要还是看集合、并发相关的东西

xxj-job 源码：这个算是自己感兴趣的

netty 源码：我又看了一边之前的文章，虽然还是没怎么看懂但是还是学到了一点东西

redisson 源码：这部分只是看了分布式锁相关的源码，看得挺乱的，目前也只是弄清楚了几个地方而已

kkfileview 源码：主要是想了解这么多文件类型的预览功能是怎么实现的

dubbo 源码：看了一点，但是没看懂放弃了
